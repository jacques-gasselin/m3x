public static final void vadd3(float[] result,

Adds the 3D vector a to the 3D vector b, storing the result in the 3D vector result.

This is equivalent to the + operator on two vectors

Note: the algorithm guarantees that using either a or b as the result vector is stable and will work as expected.

Parameters:
a - the left operand, vector.
b - the right operand, vector.
result - the vector to store the result in.
