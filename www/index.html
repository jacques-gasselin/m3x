XML Tools for M3G

This project is dedicated to providing reliable XML tools for M3G (JSR184, JSR297).

The M3G XML Schema, M3X, is a new proposal for JSR297. We aim to make a reference implementation of a reliable M3X to M3G converter using the M3X Schema. A further goal would be to also allow M3G to M3X conversion.

These tools are mainly meant for developers and implementors of M3G.

    Some quick goals of the project:
  • Define an XML Schema that can validate .m3x files
  • Use JAXB to create Marshalling facilities for .m3x files
  • Create tools to convert .m3x to .m3g
  • Create tools to convert .m3g to .m3x
